## Featured Project

### [Wiki Backend](https://github.com/Shrey-Vats/wiki-backend)

A production-grade backend system built in **Rust** using **Axum** framework.

**What it does:**  
Multi-user collaborative platform with real-time features, authentication, and resource management. Designed to handle concurrent operations efficiently.

**Architecture Highlights:**
- **Web Framework:** Axum (async Rust framework)
- **Database:** PostgreSQL with connection pooling (20 max, 2 min connections)
- **Authentication:** JWT-based with secure token generation
- **Middleware:** Custom auth middleware with cookie-based token extraction
- **Error Handling:** Comprehensive error mapping (validation, DB constraints, authorization)
- **Modular Design:** Clean separation of concerns (handlers, services, repositories)

**Key Features:**
- User management (signup, login, visibility control)
- Todo/task system with categories and tags
- Real-time collaboration rooms
- Progress tracking
- Typed error responses with meaningful HTTP status codes

**Why it matters:**  
This project demonstrates backend fundamentals — connection pooling, error handling, modular architecture, and secure authentication — all implemented at production scale.
